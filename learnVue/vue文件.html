			1.webpack安装 
			npm install --save-dev webpack -g
			npm install --save-dev webpack-cli -g
			
			2. .vue文件的学习
			第一步：cnpm install vue-loader vue-template-compiler 
					babel-plugin-transform-runtime --save-dev
				  npm install vue --save
				  (--save是安装在dependencies下面的，是运行环境要安装的包；-save-devdevDependencies是开发环境要安装的包)
			第二步：在webpack.config.js中添加babel（webpack4中使用）
				{
					test: /\.(jsx|js)$/,
					use: [{
						loader: "babel-loader",
						options: {
							presets: ['es2015', 'react']
						}
					}],
					exclude: /node_modules/
				},
			第三步：在webpack.config.js中添加
				{
					test: /\.vue$/, //解析 .vue组件页面文件
					loader: 'vue-loader' //
				},
			
			3. 将.vue中的内容解析编译并且展示在浏览器中
			  第一步：npm install vue --save
			  第二步：在main.js中编写解析.vue代码
			  (--//1. 导入vue核心包
				import vue from 'vue';
				
				//2. 导入App.vue的vue对象
				import App from './App.vue';
				
				//3. 利用vue对象进行解析渲染
				new Vue({
					el:'#app',
				//	render:function(create){create(App)}  es5的写法
					render:c=>c(App)   //es6写法
				})--)
			 第三步：需要在index1.html中写初始化页面
			 第四步：启动 npm run dev  
			 (其中遇到问题：报loader的错，因为webpack4中的配置文件有些不一样，所以将webpack.config.js中的配置改一下即可)
